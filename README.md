# 💻 Programming Fundamentals (C++)

This repository contains foundational projects developed during my **1st Semester**, focusing on core programming logic, algorithmic thinking, and C++ syntax basics.

---

## 🚀 Key Projects

### 1. Student Management System (`StudentManagementSystem.cpp`)
A console-based application designed to handle student data records.
* **Logic:** Implements basic CRUD (Create, Read, Update, Delete) operations.
* **Concepts:** Uses arrays and structural logic to manage student identities and grades.

---

### 2. Pyramid Pattern Logic (`PyramidPatternLogic.cpp`)
A demonstration of nested loop structures and mathematical coordinate logic.
* **Logic:** Features various pattern-generation algorithms to demonstrate control over loop iterations.
* **Concepts:** Mastery of `for` and `while` loops for geometric rendering in a console.

---

### 3. Fibonacci Sequence Generator (`FibonacciSequence.cpp`)
A mathematical algorithm project focused on iterative sequences.
* **Logic:** Efficiently calculates the Fibonacci series up to a user-defined limit.
* **Concepts:** Demonstrates understanding of basic recursion-like iterative logic and variable updates.

---

## 🛠️ Technical Concepts Covered

* **Control Structures:** Proficient use of `if-else` statements and `switch` cases for decision-making.
* **Iteration:** Deep understanding of nested loops for complex logic.
* **Data Types:** Proper implementation of `int`, `float`, `string`, and `bool` for memory efficiency.
* **Input/Output:** Mastery of `iostream` for interactive console applications.
